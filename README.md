# Scrape

### Overview

Scrap is a web app that lets users view and leave comments on the latest news. But instead of writing articles, flex Mongoose and Cheerio muscles to scrape news from another site.



### npm packages:

   1. express

   2. express-handlebars

   3. mongoose

   4. cheerio

   5. axios
